# Installation

AgentHiFive can be used in two modes:

## SaaS (Recommended)

Use the hosted platform at **app.agenthifive.com**. You only need to install the OpenClaw plugin — no servers, databases, or Docker required.

**Best for:** Most users, quick setup, zero infrastructure management.

→ [SaaS Setup Guide](./installation-saas.md)

## Self-Hosted

Run the full AgentHiFive stack on your own infrastructure. Gives you complete control over data, encryption keys, and networking.

**Best for:** Enterprises with strict data residency requirements, air-gapped environments, or custom security policies.

→ [Self-Host Setup Guide](./installation-selfhost.md)
